Jogo de Palavras

Como Jogar

Adivinhe a palavra de 5 letras em 6 tentativas.

Após cada tentativa, as cores das letras mudarão para mostrar o quão perto você está da resposta.

  • A - Letra correta na posição correta
  • B - Letra correta na posição errada
  • C - Letra não está na palavra

Use o teclado virtual ou o teclado do seu dispositivo para digitar.
